# 検索アルゴリズム
## 距離分の引き算について
~確かに文字数に近いものが得られるようになるかもしれないが、今回は不適切。~ <- 結局残した。
順番どおり...というか前から順に一致していったものを加点したい。
全く単語順であれば更に加点したい

+--------------------+
|条件|得点|
+--------------------+
|次のcharactorにマッチ|3点|
|次以降のcharactorにマッチ|2点|
|それ以外のcharactorにマッチ|1点|
+--------------------+
